# Week 5 - CSS Layouts & Introduction to React

This week covers advanced CSS techniques like Flexbox and the basics of building modern web applications with React.

## Files and Descriptions

- **Layout10.html**: A multi-section webpage layout using custom CSS classes and structural HTML.
- **flex.css**: A stylesheet containing Flexbox rules for responsive design.
- **flex.html**: A demonstration of Flexbox containers and item alignment.
- **flexfrontend.html**: A full frontend layout example built entirely with Flexbox.
- **react-first-app/**: A foundational React project exploring components, props, and dynamic lists.
- **stylef.css**: Styling rules specifically for Flexbox exercises.
- **stylered.css**: The design system used for Layout10 experiments.
- **stylet.css**: CSS definitions for styling HTML tables.
- **table.html**: Practice with complex HTML tables, including merging cells with `rowspan` and `colspan`.

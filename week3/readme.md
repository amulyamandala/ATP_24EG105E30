# Week 3 - Backend Development with Node.js & Express

This week introduces backend development, specifically building REST APIs using Node.js, Express, and Mongoose for MongoDB integration.

## Files and Descriptions

- **productAPI.js**: A complete Express router with endpoints for creating, reading, updating, and deleting products in a MongoDB database.
- **productModel.js**: Definition of the Mongoose schema and model for Product documents, including data validation.

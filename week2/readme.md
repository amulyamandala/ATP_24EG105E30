# Week 2 - Advanced JavaScript & Modules

Week 2 covers advanced JavaScript concepts such as higher-order array methods, the spread/rest operators, asynchronous timing, and modular code structure.

## Files and Descriptions

### day3/ - Array Methods & Data Manipulation
- **1_arraysimple.js**: Introduction to basic array operations and methods.
- **2_arraysimple.js**: Simple array iteration and basic transformations.
- **3_arraysimple.js**: Fundamental array manipulation exercises.
- **4_assignmentarrayadv1.js**: Shopping cart logic utilizing `filter`, `map`, `reduce`, `find`, and `findIndex`.
- **5_arrayadv2.js**: Advanced array transformation and filtering exercises.
- **6_arrayadv3.js**: Practicing complex data extraction from arrays.
- **7_arrayadv4.js**: Exercises on reducing arrays to single values or objects.
- **8_arrayadv5.js**: General advanced array manipulation tasks.

### day4/ - Spread/Rest Operators & Asynchronous Patterns
- **11_spreadassigment1.js**: Demonstrating array cloning and extension using the spread operator.
- **12_spreadassignment2.js**: Merging and copying objects using the spread operator.
- **13_restapi.js**: Introduction to RESTful API principles and request methods.
- **14_copy.js**: Exploring the differences between shallow and deep copying.
- **15_copy2.js**: Advanced techniques for cloning complex objects and arrays.
- **Book.js**: JavaScript class implementation for managing book collections.
- **timer1.js**: Using `setTimeout` to handle delayed execution and asynchronous tasks.
- **timer2.js**: Implementing periodic execution using `setInterval`.

### modulesassignment/ - Task Management Module
- **app.js**: Entry point for the module assignment, showcasing ES6 `import` and `export`.
- **task.js**: Logic for managing tasks within a modular environment (add, retrieve, complete operations).
- **validator.js**: Reusable validation functions for data integrity (titles, priorities, dates, etc.).

### modulesassignment2/ - E-Commerce Module
- **app.js**: Main application showcasing modular e-commerce operations.
- **product.js**: Product catalog management (fetch all products, search, filter by category).
- **cart.js**: Shopping cart operations (add items, retrieve, calculate totals, update quantities, remove items).
- **discount.js**: Discount and coupon calculation logic.
- **payment.js**: Payment processing and transaction handling.

### Root Files
- **productapi.js**: An Express-based router implementation for handling CRUD operations on an in-memory product list.

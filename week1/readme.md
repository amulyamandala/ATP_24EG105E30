# Week 1 - JavaScript Basics

This week focuses on the fundamentals of JavaScript, including functions, control flow, and basic array manipulations.

## Files and Descriptions

### 18-feb/
- **function1.js**: A function that compares three numbers and returns the largest.
- **function2.js**: General practice with JavaScript functions and logic.
- **function3.js**: Further exploration of functional programming concepts in JavaScript.
- **practice1.js**: Basic comparison logic and usage of the ternary operator.
- **practise2.js**: Introductory JavaScript exercises for beginners.
- **practise3.js**: Exercises focusing on variables and basic data types.
- **practise4.js**: Practice with console logging and basic arithmetic.

### 19-feb/
- **1_practisearray.js**: Practical examples of array methods like `splice` for adding, removing, and updating object entries in an array.
